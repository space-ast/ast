# AstMath 数学函数库

AstMath是ast项目中的数学函数库，提供与航天动力学无关的通用数学功能，包括向量、矩阵、四元数、欧拉角等的计算。

## 模块概述

AstMath模块主要提供以下功能：
- 向量运算（2D、3D、ND）
- 矩阵运算（2x2、3x3、MxN）
- 四元数运算
- 欧拉角运算
- 姿态转换（欧拉角、四元数、旋转矩阵之间的转换）
- 角度轴表示
- 旋转运算

## 核心功能

### 向量运算

提供固定维度和动态维度的向量实现

### 矩阵运算

提供固定维度和动态维度的矩阵实现


### 四元数运算

提供四元数的完整实现，用于表示旋转

### 欧拉角运算

提供欧拉角的表示和运算


### 姿态转换

提供不同姿态表示之间的转换

### 角度轴表示

提供角度轴表示的实现


## 目录结构

```
AstMath/
├── Array/              # 数组相关实现
│   ├── Vector.hpp      # 向量实现
│   ├── Matrix.hpp      # 矩阵实现
│   ├── Sparse.hpp      # 稀疏矩阵实现
│   └── MathOperator.hpp # 数学运算符定义
├── Attitude/           # 姿态表示相关实现
│   ├── Attitude.hpp    # 姿态基类
│   ├── Quaternion.hpp  # 四元数实现
│   ├── Euler.hpp       # 欧拉角实现
│   ├── Rotation.hpp    # 旋转实现
│   ├── AngleAxis.hpp   # 角度轴实现
│   └── AttitudeConvert.hpp # 姿态转换
└── README.md           # 本文档
```
